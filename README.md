# Visual-Traffic-Signal-Simulation-System

Third Semester Project for Data Structure Course

This project presents a Visual Traffic Signal Simulation System developed in C++  using the SFML (Simple and Fast Multimedia Library). The primary objective is to model  and visualize traffic flow across configurable intersection grids (1 to 4 intersections) to  demonstrate the practical application of Queue Data Structures.
